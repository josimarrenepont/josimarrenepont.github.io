# React + TypeScript + Vite

## Meu Portfólio

Este é o meu portfólio, construído com React e Vite.

### Instalação

Para instalar as dependências, use:

```bash
  npm install
```

### Execução

Para rodar o projeto em modo de desenvolvimento, use:
```bash
  npm run dev
```

### Build
Para criar uma versão de produção, use:
```bash
  npm run build
```
